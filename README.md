# Prithvin Vinod - Portfolio

A modern, interactive portfolio website showcasing software engineering projects, technical experience, and embedded systems expertise with ESP32 projects.


Visit the portfolio: [Prithvin.vercel.app](https://prithvin.vercel.app)

## ✨ Features

- **Interactive Hero Section** - Animated canvas-based introduction with smooth transitions
- **Responsive Design** - Fully responsive layout optimized for desktop, tablet, and mobile devices
- **Smooth Navigation** - Dot-based section indicators and hamburger menu for mobile
- **3D Models** - Interactive Three.js 3D model showcases
- **Animation-Heavy UI** - GSAP animations with ScrollTrigger for engaging scroll experiences
- **Dark Theme** - Modern dark color scheme with professional typography
- **Performance Optimized** - Efficient rendering and minimal dependencies

## 📋 Sections

1. **About** - Introduction and current status
2. **Experience** - Professional work and internships
3. **Projects** - Featured software projects and accomplishments
4. **ESP32 Projects** - Embedded systems and IoT projects
5. **Contact** - Ways to get in touch

## 🛠️ Technologies Used

**Frontend:**
- HTML5
- CSS3 (custom styling)
- JavaScript
- GSAP - Animation library
- Three.js - 3D model rendering

**Languages Showcased:**
- JavaScript, Python, C, C++, TypeScript
- Arduino, HTML, CSS

**Design:**
- Google Fonts (Space Grotesk, Inter, JetBrains Mono)
